使用说明：连接蓝牙

| 发送数据     | 对应操作 | state |
| ------------ | -------- | ----- |
| otherwise/无 | 混合模式 | 0     |
| r            | 校准模式 | 1     |
| R            | 静默模式 | 2     |
| m            | 绿灯追踪 | 3     |
| g            | 纯陀螺仪 | 4     |

3. 代码规范相当差劲！还好代码是我自己一个人写的嘻嘻
